# Swarm Simulator

Purpose: run local VOIDNET nodes and observe real libp2p/QUIC mesh behavior.

Two-node run:

```sh
bash labs/swarm-simulator/run-two-node.sh
```

Three-node run:

```sh
bash labs/swarm-simulator/run-three-node.sh
```

The simulator writes node state under `/tmp/voidnet-labs/swarm-simulator`.
